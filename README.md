# Матмод - лабораторная 2

1. Настроить чистый игровой цикл на Canvas (можно брать библиотеки, если хочется, но не физику из них)
2. Взять набор стирингов на выбор: (S -- steering, GS -- групповой стиринг)
3. Реализовать сцены с демонстрацией стирингов, можно все на одной, можно на разных
4. Боиды рисовать треугольниками (можно добавить спрайт, если хочется)

Наборы:

Набор А

* S-Seek (простой)
* S-PathFollowing (средний)
* S-Hide (сложный)

Набор Б
* S-Flee (простой)
* S-Pursuit (средний)
* S-WallAvoidance (сложный)

Набор В
* S-Arrival (простой)
* S-Evade (средний)
* S-OffsetPursuit (сложный)

Набор Г
* S-Wander (средний)
* GS-Cohesion (средний)
* S-ObstacleAvoidance (сложный)

Набор Д
* GS-Separation (средний)
* GS-Alignment (средний)
* Комбо: S-Pursuit + S-ObstacleAvoidance (сложный)
